### Install NodeJS with Homebrew

The missing package manager for macOS

    $ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    
    $ brew install node
    
    $ node -v
      v12.13.1
